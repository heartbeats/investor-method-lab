# 方法论分类 V2 讨论稿

更新时间：2026-03-02

## 1. 现状评估

当前分类“可展示”，但存在三个问题：

1. 同义词分裂：如“价值质量复利 / 价值+质量长期复利 / 高质量集中投资”本质接近，却被分散成多个桶。
2. 维度混杂：策略、执行、披露口径混在一个字段里，导致筛选结果不稳定。
3. 复合风格被压扁：很多投资人本质是“主策略 + 次标签”，单桶表达会丢信息。

## 2. V2 结构（草案）

V2采用三层结构：

1. `primary_family`：主策略家族（每人1个）
2. `execution_tags`：执行标签（每人多个）
3. `disclosure_tags`：披露口径标签（每人多个）

对应标准文件：

- 分类字典：[methodology_taxonomy_v2.json](/home/afu/projects/investor-method-lab/data/methodology_taxonomy_v2.json)
- 映射结果：[investor_methodology_v2_draft.json](/home/afu/projects/investor-method-lab/data/investor_methodology_v2_draft.json)
